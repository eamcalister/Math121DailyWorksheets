    Handout 9.3 Friday Week 9 - Angle Sum Identities and Sinusoidal Functions
    

    Objectives

    • Use angle sum identities to re-write sums of sinusoidal functions as single sinusoidal functions.
      🔗
      🔗
    • Use a product-to-sum identity to solve a trigonometric equation.
      🔗
      🔗
    🔗
    In class yesterday we did an activity to derive the first of the identities in the following:

    The Angle Sum Identities.

    If \(\theta\) and \(\phi\) are angles, then we have
    \begin{align*} \sin(\theta+\phi) \amp = \sin(\theta)\cos(\phi) + \cos(\theta)\sin(\phi) \\ \sin(\theta-\phi) \amp = \sin(\theta)\cos(\phi) - \cos(\theta)\sin(\phi) \\ \cos(\theta+\phi)\amp = \cos(\theta)\cos(\phi) - \sin(\theta)\sin(\phi) \\ \cos(\theta-\phi)\amp = \cos(\theta)\cos(\phi) + \sin(\theta)\sin(\phi) \end{align*}
    🔗
    🔗
    Now we will use these identities for some problems that come up in later applied math courses.
    🔗

    Example 9.5.

    In the mathematical description of oscillating objects, one frequently encounters functions of the form
    \begin{equation*} f(t) = c_1\cos(Bt) + c_2\sin(Bt), \end{equation*}
    where \(c_1\) and \(c_2\) are constants. This is a periodic function with period \(P = \frac{2\pi}{B}\text{,}\) but we would like to know its amplitude too. To do so, we can write it in the form
    \begin{equation*} f(t) = c_1\cos(Bt) + c_2\sin(Bt) = A\sin(Bt + \phi) \end{equation*}
    for some constants \(A\) and \(\phi\text{.}\) To see how, let’s do an example.
    🔗
    Let \(f(t) = 3\cos(7t) + 2\sin(7t)\text{.}\) We wish to write in the form \(f(t) = A\sin(7t + \phi)\)
    🔗
    🔗

    Summary: Writing as a Single Sine.

    To write,
    \begin{equation*} f(t) = c_1\cos(Bt) + c_2\sin(Bt) \end{equation*}
    in the form
    \begin{equation*} f(t) = A\sin(Bt + \phi) \end{equation*}
    proceed as follows:
    1. Expand \(A\sin(Bt + \phi)\) using the angle sum identity for sine.
      \begin{align*} c_1\cos(Bt) + c_2\sin(Bt) \amp = A\sin(Bt + \phi)\\ \amp = A\cos(Bt)\sin(\phi) + A\sin(Bt)\cos(\phi). \end{align*}
      🔗
      🔗
    2. Match coefficients to get
      \begin{align*} c_1 \amp = A\sin(\phi)\\ c_2 \amp = A\cos(\phi) \end{align*}
      🔗
      🔗
    3. Notice that \(A^2 = c_1^2 + c_2^2\text{,}\) so \(A = \sqrt{c_1^2 + c_2^2}\text{.}\)
      🔗
      🔗
    4. Notice that \(\tan(\phi) = \frac{c_1}{c_2}\text{.}\) Based on the signs of \(c_1\) and \(c_2\text{,}\) decide which quadrant \(\phi\) can be in, and use \(\phi = \tan^{-1}(c_1/c_2)\) or \(\phi = \tan^{-1}(c_1/c_2) + \pi\text{.}\)
      🔗
      🔗
    🔗
    🔗
    Some identities related to the angle sum identities are the following:

    Sum to Product Identities.

    If \(\theta\) and \(\phi\) are angles, then we have
    \begin{align*} \sin(a) + \sin(b) \amp = 2\sin\left(\frac{a+b}{2}\right)\cos\left(\frac{a-b}{2}\right) \\ \sin(a) - \sin(b) \amp = 2\cos\left(\frac{a+b}{2}\right)\sin\left(\frac{a-b}{2}\right) \\ \cos(a) + \cos(b) \amp = 2\cos\left(\frac{a+b}{2}\right)\cos\left(\frac{a-b}{2}\right) \\ \cos(a) - \cos(b) \amp = -2\sin\left(\frac{a+b}{2}\right)\sin\left(\frac{a-b}{2}\right) \end{align*}
    🔗
    🔗
    🔗
    One use of these is to solve equations like in the following example.
    🔗

    Example 9.6.

    Solve
    \begin{equation*} \cos(4x) + \cos(x) = 0,\ \ 0\leq x \leq \pi. \end{equation*}
